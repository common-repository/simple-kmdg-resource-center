This is the simple version of KMDG's Resource Center, allowing you to create and show your resource items inside WordPress.

== Description ==

This is the simple version of KMDG's Resource Center, intended for smaller scale solutions without needing much customization. It allows you to add resource items inside WordPress, such as webinars, PDFs, eBooks, Videos, etc.

== Frequently Asked Questions ==

= Is there a post view option =

Post view is in development at this time.

= What video sources will be supported? =

Only Vimeo and YouTube at this time.

= What about form integration? =

Form integration is included in the premium version of the plugin.

= What about other integrations? =

There will be an advanced version of the resource center being adapted for pulic use, however, it will be subscription based. Please contact us at plugin+rc@kmdg.com to discuss what you need.

= What resource types are supported? =

You can setup whatever type of resource you need, however, attachments are limited to external links and downloads.
